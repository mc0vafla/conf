📂 Навигация (Перемещение)

    k / ArrowUp — Вверх по списку

    j / ArrowDown — Вниз по списку

    h / ArrowLeft — Перейти в родительскую директорию (назад)

    l / ArrowRight — Открыть директорию / файл (вперед)

    g g — Перейти в самый верх списка

    G — Перейти в самый низ списка

    Ctrl + u — Прокрутить на пол-экрана вверх

    Ctrl + d — Прокрутить на пол-экрана вниз

    Ctrl + b — Прокрутить на страницу вверх

    Ctrl + f — Прокрутить на страницу вниз

🔍 Поиск и Фильтрация

    / — Найти файлы по имени (инкрементальный поиск, переход по n / N)

    f — Быстрый прыжок на файл по первой букве (вводишь букву и перемещаешься)

    z — Поиск и переход через утилиту zoxide (если установлена)

    w — Поиск файлов через fzf

🛠 Управление файлами (Операции)

    Space — Выделить текущий файл (переключить выделение)

    v — Включить визуальный режим выделения (как в блочных редакторах, выделение стрелочками)

    V — Снять выделение со всех файлов

    y — Скопировать выбранные файлы (Yank)

    x — Вырезать выбранные файлы (Cut)

    Y — Скопировать только путь к файлу (p — абсолютный, d — только папку, n — только имя)

    p — Вставить скопированные/вырезанные файлы (Paste)

    d d — Переместить файл в корзину (Trash)

    D — Удалить файл навсегда (Полноценное удаление)

    a — Создать новый файл (или папку, если имя заканчивается на /)

    r — Переименовать файл

    ; a — Смена прав доступа (chmod)

🖥 Вкладки (Tabs)

    t — Создать новую вкладку (откроется в текущем пути)

    w — Закрыть текущую вкладку

    1 .. 9 — Перейти на вкладку с соответствующим номером

    [ — Переключиться на предыдущую вкладку

    ] — Переключиться на следующую вкладку

⚙️ Отображение и Сортировка

    . — Переключить показ скрытых файлов (файлы на точку)

    m — Изменить режим сортировки (по имени, размеру, времени модификации и т.д.)

    Ctrl + p — Скрыть/показать правую панель предпросмотра

🚀 Запуск и Задачи

    Enter — Открыть файл в приложении по умолчанию

    o — Меню «Открыть с помощью...» (выбор программы)

    : — Вызвать командную строку Yazi для ручного ввода команд

    w — Открыть текущий путь в системном терминале

    ~ — Открыть панель текущих фоновых задач (Tasks, например, процесс копирования больших файлов)

    q — Выйти из Yazi
